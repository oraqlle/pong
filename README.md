## Building

### Perquisites

- [SFML](https://www.sfml-dev.org/download.php)
- [bpt](bpt.pizza/docs/latest/index.html)

```sh
# Build using `gcc.yaml`
bpt build -t gcc.yaml -o build
```

## Running

```sh
# Launch application
./build/bin/pong
```
